# Sobre o repositório
Esse respositório tem como objetivo recriar um jogo da velha em ruby com interface via terminal.
O repositório foi baseado em um código legado não versionado, localizado em: `legacy/tictactoe.rb`

# Instalação (Soon)
Run `bin/setup`

# Rodar
Run `bin/start`

# Tests
Run `bin/rspec`
